# Viaje Europa · Familia García 🌊🫒🏛️

PWA mobile-first, 100% offline, para el viaje familiar a Europa — junio 2026.

## Stack
- React + TypeScript + Vite
- vite-plugin-pwa + Workbox (offline)
- Dexie.js (IndexedDB)
- React Router DOM

## 3 temas visuales
| Tema | Destino | Color |
|------|---------|-------|
| `greece` | Santorini | Azul #1B3FA6 |
| `puglia` | Alberobello | Terracota #C2622E |
| `cruise` | Adriático | Marino #1A5FAA |

## Desarrollo
```bash
npm install
npm run dev
```

## Deploy
Conectar repo a Vercel → deploy automático.
